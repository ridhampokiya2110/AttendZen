# AttendZen - Attendance Management System

AttendZen is a modern, user-friendly attendance management system built with Next.js and TypeScript. It helps students and professionals track their attendance across different subjects or courses.

## Features

- Track attendance for multiple subjects
- Real-time attendance percentage calculation
- Visual attendance analytics with charts
- Target attendance tracking
- AI-powered attendance tips
- Responsive design for all devices
- Dark mode support

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: Shadcn/ui
- **Charts**: Custom charting components
- **State Management**: React Hooks

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:9002](http://localhost:9002) in your browser

## Usage

- Click "Add Subject" to create a new subject
- For each subject, you can:
  - Mark attendance (Present/Absent)
  - Set attendance targets
  - View attendance statistics
  - Delete subjects when needed
